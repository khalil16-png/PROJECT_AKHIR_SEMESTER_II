## 🚀 Cara Menjalankan

1. **Buka langsung** file `index.html` di browser, atau
2. **Gunakan Live Server** di VS Code untuk development experience yang lebih baik

---

## 📸 Upload Foto & Screenshot

Letakkan file gambar di folder yang sesuai:

| File | Lokasi | Keterangan |
|------|--------|------------|
| Foto profil | `assets/img/profile.jpg` | Format JPEG/PNG, rasio 1:1 disarankan |
| Screenshot To Do List | `assets/img/projects/todo-app.jpg` | Screenshot dari demo website |
| Screenshot Poster | `assets/img/projects/poster-ramadhan.jpg` | Export poster dari Canva |

---

## 🌐 Deploy ke GitHub Pages

1. Push folder ini ke GitHub repository
2. Pergi ke **Settings → Pages**
3. Source: **Deploy from a branch → main → / (root)**
4. Tunggu beberapa menit → website live!
